# wxcloud-databases-import

> 基于electron-vue实现的微信小程序云开发数据库文件批量导入。

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
